# zakazkomat
Scrapy example project - programming contract demands scraping

Available spiders:
  * webtrh.cz
  * epoptavka.cz
  * webdeal.cz
  
Spiders crawl though software development demand listings only.

## Installation
1. Clone repository
```
$ git clone ...
```
2. Open repository and install requirements
```
$ cd zakazkomat
$ pipenv install
```
3. Open scrapy directory and have fun
```
$ cd zakazkomat
$ scrapy list
$ scrapy crawl SPIDERNAME
```
